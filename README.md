# 🏛️ Totem Interativo de Museu – APS (UNIP)

##  Sobre o Projeto
Este projeto foi desenvolvido como parte da Atividade Prática Supervisionada (APS) do curso de Ciência da Computação da UNIP (1º semestre de 2026).

A proposta consiste na criação de um totem interativo para um museu multitemático, com foco em uma exposição sobre robôs exploradores de Marte.

---

##  Objetivo Geral
Desenvolver um sistema interativo que auxilie visitantes de um museu a:
- Consultar obras da exposição
- Visualizar informações sobre cada item
- Responder a um questionário interativo

---

##  Objetivos Específicos
- Identificar necessidades dos usuários e traduzi-las em soluções
- Aplicar conceitos de programação orientada a objetos
- Desenvolver habilidades em trabalho em equipe
- Projetar uma interface intuitiva para uso em telas sensíveis ao toque

---

##  Funcionalidades

###  Exposição
- Exibição de 10 obras (imagens)
- Apresentação de um breve histórico de cada obra

###  Questionário Interativo
- Questionário com 5 perguntas de múltipla escolha
- Exibição do resultado ao final
- Dados armazenados temporariamente (sem persistência em disco)

###  Processamento de Dados
- Armazenamento em estruturas dinâmicas (vetores)
- Possibilidade de geração de relatórios para análise

---

##  Tecnologias Utilizadas
- Java
- JDialog (Swing)
- Programação Orientada a Objetos (POO)

---

##  Requisitos
- Interface adaptada para tela touchscreen
- Implementação de teclado virtual próprio
- Não utilizar teclado do sistema operacional
- Não coletar dados sensíveis (LGPD)

---

##  Restrições
- Sem persistência de dados em disco
- Proibido uso de dados pessoais dos usuários
- Proibido plágio (total, parcial ou conceitual)
